## 프로젝트 실행 방법

1. clone 후 install & start

```bash
npm install
npm start
```

2. 홈 화면에서 로그인/또는 회원가입 선택

3. 회원가입 성공 시 로그인 페이지로 이동

4. 로그인 성공 시 할 일 입력 페이지로 이동
